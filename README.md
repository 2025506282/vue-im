## Download
```
git clone https://github.com/Aaaaaaaty/vue-im
```
## Deploy
#### Client
```
npm install
```
#### Server
```
npm install -g nodemon //[node's self-restart tool](https://nodemon.io/)
```
#### Database
```
brew install mongodb

```
## Run
### Client
```
npm run dev
```
### Server
```
nodemon ./server/bin/www
```
### Database
```
sudo mongod --dbpath=/data/db --port=27017
```
